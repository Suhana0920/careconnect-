# CareConnect+ 🏥

A web-based appointment booking and healthcare management system built with Python and Django.

## 🌟 Features
- Patient registration and login
- Doctor listing and selection
- Smart appointment booking with time slots
- Patient dashboard
- Doctor dashboard
- Admin dashboard
- Role-Based Access Control (RBAC)
- Patient medical records
- Live chat and consultation

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Database:** MySQL, SQLite
- **Frontend:** HTML, CSS, JavaScript, Tailwind CSS
- **Tools:** Git, GitHub, VS Code

## 👥 User Roles
- **Patient** — Book appointments, view medical records, AI chat
- **Doctor** — View appointments, manage patients
- **Admin** — Full system access, manage all users and appointments

## 🚀 How to Run Locally
```bash
# Clone the repo
git clone https://github.com/Suhana0920/careconnect-.git

# Install dependencies
pip install -r requirements.txt

# Run server
python manage.py runserver
```
